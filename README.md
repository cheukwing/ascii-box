# Ascii-Box

Draws a box with given width and height to the terminal, e.g:

```
$ python3 box.py 4 4

┌--┐
|  |
|  |
└--┘

```

## Requirements

- [Python3.6](https://www.python.org/downloads/)

## Running

To run:

```shell
$ python3 box.py <width> <height>
```

To run tests:

```shell
$ python3 tests.py
```
